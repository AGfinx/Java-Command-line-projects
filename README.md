**Features**

* Add Student

* Delete Student
* Display All Students

**Features**

# ▶️ How to Run Any Project

1️⃣ Navigate into the project folder:

```bash id="cmd1"
cd project-name
```

2️⃣ Compile:

```bash id="cmd2"
javac Main.java
```

3️⃣ Run:

```bash id="cmd3"
java Main
```
